# A Regional Self-Training Framework for Binary Classification of Intrusion Detection under Class Imbalance

本專案為碩士研究成果，開發一套**區域自訓練框架 (Regional Self-Training Framework)**。旨在解決入侵偵測系統 (IDS) 在現實環境中面臨的兩大瓶頸：**標記資料極度稀缺 (僅 1%)** 與**嚴重的類別不平衡**（惡意攻擊流量遠少於正常流量）。

本框架將標準的自訓練 (Self-Training) 流程進行重構，引入「密度感知區域劃分」與「針對性區域過採樣」技術，顯著提升模型對隱匿攻擊的識別能力。

**核心目標**：在極低標註比例下，強化入侵偵測系統對於少數類別（惡意攻擊）的偵測魯棒性。

---

## 為什麼需要改良 Standard Self-Training？
原始的自訓練 (Standard Self-Training) 依靠模型自身的高信心度預測作為偽標籤 (Pseudo-labels)，但在入侵偵測場景下存在以下挑戰：

1. **類別偏誤放大 (Majority Class Bias)**：無標籤資料中正常流量佔絕大多數，導致生成的偽標籤極易向多數類別傾斜，抑制模型學習攻擊特徵。
2. **錯誤標籤累積 (Error Accumulation)**：受限於初始標記不足，模型初期的偏誤預測會轉化為錯誤標籤並參與後續迭代，導致模型效能崩潰。
3. **忽略資料分佈結構 (Ignoring Structural Patterns)**：僅依賴 Softmax 信心度門檻，無法區分特徵空間中決策邊界上的模糊樣本，容易將雜訊引入訓練集。

---

## 核心優化策略 I：密度感知偽標籤生成 (Density-Aware Strategy)

本框架引入 **HDBSCAN 密度感知分群** 與 **KNN 一致性檢查**，從空間結構面確保偽標籤的可靠性。

### 1. 關鍵發現：利用局部密度優化區域決策 (HDBSCAN)
* **技術邏輯**：利用 HDBSCAN 動態調整分群邊界，根據局部密度變化將資料劃分為不同純度的「區域 (Regions)」。
* **優化機制**：
    * **區域化標註**：將叢集分為**高純度**（採多數決標註）、**中純度**（依模型信心度標註）與**低純度**（直接捨棄），避免在混合區域產生雜訊。
    * **捕捉結構先驗**：攻擊流量在特徵空間中往往呈現特定的密度分佈，透過分群能有效鎖定潛在的惡意叢集。

### 2. 標籤校驗：KNN Consistency Check
* **技術邏輯**：對所有生成的偽標籤樣本進行 5-NN 鄰近標籤一致性檢查。
* **優化效益**：若樣本與其鄰居標籤存在嚴重衝突，則視為潛在雜訊並剔除。這能有效過濾邊界區域的誤標樣本，確保 Retraining 階段的資料品質。

---

## 核心優化策略 II：區域性過採樣 (Regional Oversampling)

針對少數類別樣本稀疏問題，本專案重新設計了針對區域特性的數據平衡邏輯：

### 1. 區域性過採樣 (In-Cluster SMOTE)
* **功能**：限制 SMOTE 僅在特定的同質叢集 (Clusters) 內部生成合成樣本。
* **巧思**：解決傳統全局 SMOTE 容易跨越決策邊界產生「不真實樣本」的痛點。透過分群限制，確保合成樣本符合局部的物理分佈邏輯。

### 2. 比例自適應平衡 (Proportional Augmentation)
* **功能**：在高純度攻擊叢集中，根據全局類別分佈比例動態計算合成倍率。
* **巧思**：在不引入背景雜訊的前提下，精準填補攻擊樣本的空間空隙，顯著強化模型對稀有攻擊行為的特徵捕捉能力。

## 實驗成果與關鍵數據

本框架於兩大主流資安資料集進行驗證。在僅使用 **1% 標註樣本** 的極端環境下，效能表現顯著優於其他基準方法：


### 1. UNSW-NB15 資料集結果 (1% Labeled)
| 方法 | Precision | Recall | F1-score | Accuracy |
| :--- | :---: | :---: | :---: | :---: |
| **Standard-ST** | 92.7% | 94.5% | 93.6% | 98.4% |
| **SSET-MTC (2024)** | 95.3% | 95.4% | 95.3% | 95.9% |
| **本研究提出方法 (Ours)** | **95.6%** | **98.0%** | **96.8%** | **99.2%** |

### 2. CICIDS2017 資料集結果 (1% Labeled)
| 方法 | Precision | Recall | F1-score | Accuracy |
| :--- | :---: | :---: | :---: | :---: |
| **Standard-ST** | 91.6% | 93.9% | 92.7% | 98.4% |
| **FixMatch** | 93.9% | 93.6% | 93.8% | 92.9% |
| **本研究提出方法 (Ours)** | **95.0%** | **97.8%** | **96.4%** | **99.2%** |


### 3. 消融實驗分析 (Ablation Study on UNSW-NB15) 
為了驗證各組件的貢獻度，我們進行了組件拆解測試：

| 配置方案 | 組件內容 | F1-score | 關鍵貢獻 |
| :--- | :--- | :---: | :--- |
| **Orig-ST** | 基準自訓練法 | 93.60% | 基礎效能  |
| **DAP-only** | 僅使用密度感知偽標籤 | 95.73% | 提升精確率 (Precision) |
| **RO-only** | 僅使用區域性過採樣 | 96.08% | 大幅提升召回率 (Recall) |
| **All w/o KNN** | 組合技術（不含 KNN 校驗）| 96.30% | 整合優勢初現  |
| **ALL (Ours)** | **完整區域自訓練框架** | **96.81%** | **達到最佳平衡與最高性能** |

### 實驗發現
* **對比 Standard ST**：整體 F1-score 在兩個資料集上分別提升了 **3.2%** 與 **3.7%**。
* **少數類別強化**：攻擊樣本的召回率 (Recall) 顯著增長，證明區域增強策略能有效降低漏報率。
* **優於現有技術**：效能表現全面超越 **SSET-MTC (2024)** 與 **FixMatch** 等先進半監督學習框架。

## 專案貢獻
1. 提出**密度感知偽標籤策略**，有效解決自訓練過程中的多數類別偏誤問題。
2. 開發**區域性過採樣機制**，在保持數據結構完整性的同時強化失衡資料的學習。
3. 實證在**極度少量標籤 (1%)** 環境下，仍能建立高效能、低漏報的入侵偵測系統。
